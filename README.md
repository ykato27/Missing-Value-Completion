# missing-value-completion
* 基本的な欠損値補完の手法

## リポジトリ構成
```
.
├── README.md                 READMEファイル
├── .dockerignore        
├── Dockerfile                Dockerファイル
├── docker-compose.yml
├── notebook                  jupyter notebook
└── data                      dataファイル
```

## 環境構築

* Dockderfileがあるホスト側のフォルダへ移動（例：Desktop/missing-value-completion）
```
cd Desktop/missing-value-completion
```

* Dockerによる環境構築（フォルダをマウント：Desktop/missing-value-completion）
```
docker-compose up --build
```

* ブラウザーを立ち上げてlocalhost:8888へアクセス
* ローカルフォルダがマウントされている

## jupyter notebook説明
* missing-value-completion.ipynb : 欠損値補完を実装したnotebook
